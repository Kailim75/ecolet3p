📝 Instructions d'impression

  • Format : 85 × 55 mm (standard européen)
  • Papier : 350g/m² couché mat ou satiné
  • Impression : Recto/verso, couleurs CMJN
  • Finition recommandée : Pelliculage mat soft-touch
  • Quantité minimum : 250 exemplaires
💡 Astuce : Activez "Graphiques d'arrière-plan" dans les paramètres d'impression (Ctrl+P) pour conserver les couleurs du fond.

🎯 Philosophie de design V3

Une carte = Une action. Cette version épurée supprime toute friction cognitive. Le recto présente l'identité professionnelle sans distraction. Le verso guide vers un seul objectif : envoyer un message WhatsApp. Le QR code pré-remplit le message pour maximiser le taux de conversion.
