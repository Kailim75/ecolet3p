📋 Guide d'export PDF pour impression professionnelle

  • Format : A5 Portrait (148 × 210 mm)
  • Qualité : Haute résolution 300 DPI
  • Couleur : CMJN recommandé
  • Papier : 170-250g/m² couché brillant
  • Marges : Fond perdu 3mm inclus
  • Design : Optimisé psychologie cognitive

🔧 Comment exporter chaque flyer en PDF :

  1. Cliquez sur "Ouvrir" pour le flyer souhaité
  2. Utilisez Ctrl+P (ou Cmd+P sur Mac)
  3. Sélectionnez "Enregistrer au format PDF"
  4. Format : A5 Portrait
  5. Cochez "Graphiques d'arrière-plan"
  6. Marges : "Aucune" ou "Minimum"
