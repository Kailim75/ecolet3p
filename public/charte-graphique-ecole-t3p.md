## Règles Typographiques

### ✅ À faire

1. **Titres** : Toujours en Plus Jakarta Sans, couleur `#1B4D3E`
2. **Corps** : Toujours en Inter, couleur `#5A5A5A` ou `#4B463C`
3. **Hiérarchie** : Respecter l'ordre H1 > H2 > H3 (pas de saut)
4. **Longueur de ligne** : Maximum 75 caractères par ligne
5. **Interlignage** : Minimum 1.5 pour les paragraphes

### ❌ À éviter

1. Ne JAMAIS utiliser d'autres polices (Arial, Helvetica, Times...)
2. Ne JAMAIS mettre les titres en minuscules simples
3. Ne JAMAIS utiliser des tailles intermédiaires non définies
4. Ne JAMAIS dépasser 4 niveaux de hiérarchie sur une page

---

# 8. RÈGLES DE COHÉRENCE

## ✅ Ce qu'il faut TOUJOURS faire

| Règle | Détail |
|-------|--------|
| **Utiliser les couleurs exactes** | Toujours les codes HEX définis, jamais "à peu près" |
| **Respecter la hiérarchie typo** | H1 > H2 > H3, pas de saut de niveau |
| **Maintenir les contrastes** | WCAG AA minimum (4.5:1 texte, 3:1 éléments UI) |
| **Utiliser des coins arrondis** | Cohérence avec le design system (12-16px) |
| **Appliquer les ombres douces** | Pas d'ombres dures ou brutales |
| **Centrer les CTA** | Les appels à l'action doivent être visibles |
| **Inclure les mentions légales** | SIRET, agrément sur tous les documents |
| **Utiliser des photos réelles** | Du centre, des promotions, de l'équipe |

## ❌ Ce qu'il faut ÉVITER

| Règle | Détail |
|-------|--------|
| **Ne jamais déformer le logo** | Ni étirer, ni comprimer, ni pivoter |
| **Ne pas utiliser d'autres couleurs** | Pas de bleu, rouge, violet comme primaires |
| **Éviter les fonds chargés** | Pas de textures, motifs complexes |
| **Ne pas mélanger les polices** | Pas de Roboto, Arial, Times avec la charte |
| **Éviter les effets excessifs** | Ombres dures, dégradés arc-en-ciel, néon |
| **Ne pas réduire les espaces** | Respecter les marges minimales |
| **Ne pas utiliser de photos stock** | Pas de photos génériques d'acteurs |
| **Éviter le ton marketing agressif** | Pas de "INCROYABLE", "EXCEPTIONNEL" |

## Critères d'une Image Professionnelle et Crédible

| Critère | Comment l'atteindre |
|---------|---------------------|
| **Sérieux** | Couleurs sobres, pas de fantaisie |
| **Expertise** | Chiffres visibles (94%), certifications |
| **Confiance** | Photos réelles, mentions légales |
| **Clarté** | Hiérarchie visuelle forte, peu de texte |
| **Accessibilité** | Contrastes élevés, tailles lisibles |
| **Cohérence** | Même style partout (site, print, RS) |
